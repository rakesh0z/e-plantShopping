# e-Plant Shopping

This is a simple React + Redux shopping cart application for plants. It includes:

- Product listing with categories
- Add-to-cart functionality
- Cart console for quantity updates and item removal
- Redux Toolkit state management

## Getting Started

### Install

```bash
npm install
```

### Run in development

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Redux

- Store is configured in `src/store.js`
- Cart slice logic is in `src/CartSlice.jsx`

## Deployment (GitHub Pages)

If you want to deploy to GitHub Pages, you can add `gh-pages` scripts in `package.json` and set the correct `base` path in `vite.config.js`.

## Notes

This project is in active development and is used for learning React/Redux patterns.
